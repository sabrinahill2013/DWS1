Element Definitions

h1
A first-level heading used for main headings.
h2
A second-level heading used for sub headings.
h3
A third-level heading. Third most important heading.
h4
A fourth-level heading. Forth most important heading.
h5
A fifth-level heading. Fifth most important heading.
h6
A sixth-level heading. Sixth most important heading.
p
Used to indicate a paragraph that include one or more sentences.
b
Used to make text on a webpage appear bold.
i
Used to make text on a page appear in italics.
sup
Used to indicate a superscript. Mainly used for suffixes.
sub
Used to indicate a subscript such as footnotes.
br
Used to place a line break on a web page.
hr
This is an empty element with one tag before the closing angled bracket.
strong
Used to show strong importance in a word.
em
Shows emphasis that subtly changes. Most likely will appear in italics.
blackquote
Used for long quotations that are the size of a paragraph.
q
Used for quotes that are short and reside within a paragraph.
abbr
Used when using an abbreviation or acronym.
cite
Used to reference a source of information such as a web page or book. Not used for people's name.
dfn
Used to show new terms and show the defining instance for the new terms.
address
Used to contain the contact information of the author or a page.
ins
Used to indicate where content has been inserted.
del
Used to show where content has been taken out.
s
This is a strikeout. Used to show where information is no longer correct.
ol
This is an ordered list. This list will be numbered.
li
This is used to indicate a list item.
ul
This is an unordered list that will show bulleted.
dl
This is used to indicate a list of definitions or terms.
dt
This is used to name the term that will be defined.
dd
This is used to define the term within the definition term tag.
